# Formação Lua - DIO

Este repositório contém exercícios, projetos e desafios da formação em linguagem Lua oferecida pela Digital Innovation One (DIO).

## Estrutura do Repositório

- `dicionarios.lua`: Exercício sobre dicionários (tabelas) em Lua.
- `geradorSenhas.lua`: Projeto de gerador de senhas aleatórias, utilizando letras, números e símbolos.
- `listas2.lua`: Exercício sobre listas (tabelas como arrays) em Lua.
- `DIO - Trilha lua/`: Pasta com módulos da trilha de aprendizado.
  - `Módulo 1/`: Conteúdo introdutório.
    - `Desafio do módulo/`: Desafio prático do módulo 1.
      - `desafio.lua`: Código do desafio.
  - `Módulo 2/`: Conteúdo avançado.
    - `avançando na programação/`: Tópicos avançados em programação Lua.
      - `main.lua`: Código principal do módulo 2.

## Pré-requisitos

- [Lua](https://www.lua.org/download.html) instalado no sistema (versão 5.1 ou superior recomendada).
- Um editor de texto ou IDE que suporte Lua (ex.: VS Code com extensão Lua).

## Como Executar

1. Clone ou baixe este repositório.
2. Navegue até a pasta do projeto.
3. Execute os arquivos Lua com o comando `lua <nome_do_arquivo.lua>`.

Exemplo:
```
lua geradorSenhas.lua
```

Para o gerador de senhas, digite o número de caracteres desejado quando solicitado.
